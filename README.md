# linux-commands-notes
This repository contains basic Linux commands and their usage for beginners.

# 🐧 Linux Basic Commands
ls = list files in directory  
pwd = show current directory path  
cd = change directory  
mkdir = make new directory  
rmdir = remove empty directory  
touch = create new file  
rm = delete file  
cp = copy file  
mv = move or rename file  
cat = show file content  
echo = print text on screen  
whoami = show current user  
clear = clear terminal screen  
man = show command manual/help  
sudo = run command as admin 

# File Management commands
ls -la = show all files (including hidden) in long format
cd .. = go one directory back
cd ~ = go to home directory
find = search files and directories
locate = find files quickly using database
grep = search text inside files
chmod = change file permissions
chown = change file owner
df -h = show disk space usage
du -sh = show size of folder/file
ps = show running processes
top = show live system processes
kill = stop a process using PID
uname -a = show system information
ifconfig = show network details (old method)
ip a = show IP address and network interfaces
wget = download file from internet
curl = fetch data from URL
history = show previously used commands
